My List Operations

This project demonstrates basic list operations in Python. It includes creating a list, appending elements, inserting values, extending the list, removing elements, sorting, and finding the index of a specific value.

Features

- Create an empty list
- Append elements to the list
- Insert an element at a specific position
- Extend the list with another list
- Remove the last element from the list
- Sort the list in ascending order
- Find and print the index of a specific value

Installation

To run this project, ensure you have Python installed on your machine. You can download Python from [python.org](https://www.python.org/downloads/).

1. Clone this repository to your local machine:

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:

   ```bash
   cd path/to/your/project
   ```

3. Run the program:

   ```bash
   python my_list.py
   ```

Code Explanation

The following operations are performed in `my_list.py`:

1. Create an empty list called `my_list`.
2. Append the values `10`, `20`, `30`, and `40` to `my_list`.
3. Insert the value `15` at the second position (index `1`).
4. Extend `my_list` with another list: `[50, 60, 70]`.
5. Remove the last element of `my_list` (which will be `70`).
6. Sort `my_list` in ascending order.
7. Find and print the index of the value `30` in `my_list`.

Output

The program will output the index of the value `30` and the final sorted state of the list. For example:

```
The index of the value 30 in my_list is: 3
Final my_list: [10, 15, 20, 30, 40, 50, 60]
```
